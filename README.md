# 환불도우미 - Backend B

AI 기반 계약서 분석 및 환불 요청 문구 자동 생성 서비스입니다.

---

## 구현 기능

### 1. 부당계약서 의심알림
계약서 내용을 입력하면 AI가 소비자 권리를 침해하는 부당 조항을 자동으로 탐지합니다.

- 탐지 대상: 환불 불가 조항, 법정 한도 초과 위약금, 자동 연장 조항 등
- 출력: 위험도(높음/보통/낮음), 부당 조항 목록, 근거 법령, 총평

### 2. 환불 요청 문구 생성
결제 정보와 환불 금액을 입력하면 상황에 맞는 문구 3종을 자동 생성합니다.

- 카카오톡/문자용: 간단한 2~3문장
- 이메일용: 정식 공문 형식
- 강경대응용: 법조문 인용 + 소비자원 신고 예고 포함

> 환불 금액 계산(환불예상금, 부당위약금)은 Backend A 담당이며, Backend B는 그 결과를 받아 문구를 생성합니다.

## 로컬 실행 방법

**1. 의존성 설치**
```bash
cd backend-b
pip install -r requirements.txt
```

**2. API 키 설정**
```bash
python setup.py
```
[Groq 콘솔](https://console.groq.com)에서 발급받은 키(`gsk_`로 시작)를 입력하면 `.env` 파일이 자동 생성됩니다.

**3. 서버 실행**
```bash
uvicorn main:app --reload
```

**4. 테스트 화면 접속**

브라우저에서 `http://127.0.0.1:8000` 접속

## API 엔드포인트

### 서버 상태 확인
```
GET /api/b/health
```

### 계약서 분석
```
POST /api/b/analyze/contract
```
```json
{
  "계약서_텍스트": "본 계약은 환불이 불가합니다. 위약금은 잔여금액의 50%입니다.",
  "업종": "헬스장"
}
```

### 환불 문구 생성
```
POST /api/b/generate/message
```
```json
{
  "업종": "헬스장",
  "업체명": "강남헬스클럽",
  "결제금액": 300000,
  "이용일수": 12,
  "환불예상금": 240000,
  "부당위약금": 60000
}
```

전체 요청/응답 명세는 [`backend-b/API_DOCS.md`](backend-b/API_DOCS.md)를 참고하세요.
